# Archive

Historical planning and audit documents from the build-out of this app,
kept for provenance. They predate the **Spellwright → PipGlyph rebrand**
(June 2026), so brand names, roadmaps, and "planned" features inside them
are snapshots of their moment — not descriptions of the current product.
Everything they planned (phases 10–11, the creation audit follow-ups, the
v2 refactor) has shipped.

| Document | What it was |
| --- | --- |

| `11_PREMIUM_POLISH/` | Phase 11 chunk specs (01–16) — gallery polish, billing, themes, e2e scaffolding. |
| `12_CREATION_AUDIT.md` | The 2026-06-09 creation-audit findings + competitor research. |
| `REFACTOR_NOTES.md` | v2 refactor session notes and branch reconciliation. |
